# Getting Started with Igniter

Igniter is an accountability-first language platform.

## Installation

Install the Igniter toolchain using the standard package manager:

```shell
gem install igniter
```

## Your First Contract

Contracts define the **behaviour** of your program:

- Declare inputs and outputs
- Specify types and constraints
- Compose with other contracts

See the [tutorial](/en/tutorial/) for a complete walkthrough.
