§01

The Spark

six arms · one hot core
arm ×6 core ⌀18 pearl r3.8

It's the moment something catches. Six arms throwing off heat, a white-hot center, a calm pearl at the middle that says the fire is under control.

Igniter takes loose intent and ignites it into a typed, observable program. The mark is that ignition, made simple enough to draw from memory: an asterisk that also happens to be a star, a spark, and — if you squint — a tiny огонёк. No diagram to decode, no story to read. Just warmth and precision in the same shape.

The six armsIgnition radiating outward. Always Ignition orange, always rounded — never sharp, never cold.
The hot coreA small ember gradient. The brightest point in the whole system — earned, used once.
The pearlA still white center. Control. The reason you trust a flame you built on purpose.

Wordmark & Casing

Specimen — set in a compiler's type
igniter-lang
use igniter::lang;

The :: path-form is the code-native alternate — for CLIs, imports and headers. The hyphen always carries the ignition color.

Casing & usage rules
✓
igniter-lang — lowercase, hyphenated. The canonical wordmark.
✓
igniter::lang — path-form alternate for code & technical surfaces.
✓
Hyphen always in ignition color.
✕
Igniter Lang — never title-case, never spaced.
✕
IgniterLang — never camel-case or run together.
✕
Never set the wordmark in a sans or serif — it's mono, always.
§08

Voice

precise, but warm
✓ Sounds like us
→

Plain over clever. "Every value carries a receipt." Not "epistemically-witnessed provenance envelopes."

→

Talk to a person. Second person, present tense. "You declare it; the compiler proves it."

→

Confident, not cold. We're sure of the work, so we don't have to shout. A little warmth is allowed.

→

Show the code. The fastest way to explain Igniter is a six-line .ig snippet.

✕ Not us
→

Hype and superlatives. No "revolutionary", "blazing-fast", "game-changing".

→

Jargon walls. If a sentence needs a glossary, rewrite the sentence.

→

Emoji confetti or exclamation storms. One honest spark is the loudest we get.

→

Talking down. Our readers write compilers for fun — respect that.
