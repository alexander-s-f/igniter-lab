The buttons attach a host-generated HostActionRecord JSON (the exact shape FrameBindingEffectResult::to_host_action_json emits from a real P18 bridge run) to the latest frame — plain data, no machine, no secret. The lineage panel then shows the action + receipt.

The console invents no layout primitives — it reuses igniter-frame's hit-test for its chrome and embeds the target's own SVG. No igniter-machine symbols, localhost only. Host maps DOM events; Rust owns the target runtime, recording, time-travel, and diff.
