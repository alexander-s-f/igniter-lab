Layout playground — author a screen as text

Type a layout in the box. Each line is <kind> <id> [fixed N | flex N] [pad N] [gap N] [cross N] [align start|center|end], where kind is col / row / leaf; two spaces of indent nest a child. cross N gives a fixed cross-axis size and align places such children on the cross axis. The Rust layout::parse + solve run in your browser on every keystroke and draw the solved boxes — deterministic, machine-free. This is the text surface the layout vocabulary opens (the engine behind list / table / form).

Boxes are tinted by nesting depth and labeled by id. fixed N takes N px on the main axis; flex N shares the rest by weight (remainder spread one-px-each, so it fills exactly). The cross axis stretches. Resize a fixed or change a flex weight and watch everything reflow.
