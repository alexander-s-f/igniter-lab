A fully Igniter-authored game — logic + view on the VM

Every frame here was produced by two .ig contracts run on igniter-vm: Step(world, boom) → world (the integer physics) and View(world) → Scene (the 3D→2D projection). The host (frame-ui, in wasm) does the ONLY non-.ig work: it renders the projected scene. Both contracts are bit-identical to a Rust reference (cross-checked); replay + time-travel are free from determinism.
