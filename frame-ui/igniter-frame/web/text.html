Keyboard text entry

Click a field, then type. Backspace deletes · Enter sends. The pointer derives a focus intent (hit-test); each keystroke is routed as a system intent through the Rust runtime's send("type"/"backspace"/"submit") — no runtime change, the reducer edits whichever field is focused. Every keystroke is a deterministic, content-addressed frame.
