Deterministic game loop — replay + time-travel

Six bodies bounce in a 3D box under integer physics. Click the scene (or 💥) to fire a boom impulse. The world at any tick is a PURE function of (initial, input log, tick) — so replay reruns the same inputs bit-for-bit, and the time-travel slider scrubs to any past/future tick by re-running the deterministic sim. No snapshots, no f64, no clock.
