Язык

Igniter — это язык программирования, ориентированный на подотчётность. Язык активно развивается — API и артефакты могут изменяться.

Обзор

Язык строится на трёх принципах:

  • Контракты — каждая сигнатура функции — верифицируемый контракт.
  • Доказательства — поведение доказывается локально до деплоя.
  • Компонуемость — модули компонуются без скрытых зависимостей.
